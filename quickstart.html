Quickstart¶

Installation¶

Download the latest stable version of lucullus_rest from GitHub. and unzip it.

Use “cd” in your console to move to the new unzipped folder like this.

cd C:\Users\user\Downloads\lucullus_rest-main\lucullus_rest-main

Finally install lucullus_rest with the following command. This installs lucullus_rest for your current user with all its dependent Python libraries.

python setup.py install --user
